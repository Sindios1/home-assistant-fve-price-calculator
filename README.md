# FVE SPOT Revenue CZ (pro Home Assistant)

Tuto integraci snadno stáhnete a nainstalujete přes HACS jako **Vlastní repozitář (Custom Repository)**.

## Instalace přes HACS

1. Otevřete HACS ve vašem Home Assistantu.
2. V pravém horním rohu klikněte na tři tečky a vyberte **Vlastní repozitáře (Custom repositories)**.
3. Do pole `Repozitář` zkopírujte odkaz na tento váš GitHub repozitář.
4. Do pole `Kategorie` vyberte **Integrace**.
5. Klikněte na Přidat. Následně integraci klasicky stáhněte a restartujte Home Assistant.
